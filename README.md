# Let-s-Work-Out

- 專題簡述：一個為健身朋友們打造的平台網站，使用者可瀏覽各種訓練課程，及觀看教練名單簡介，決定哪個課程後便可下單購買該課程，在商城也買到各種器材及營養食品， 註冊會員後可修改會員資料、查詢訂單等 。
- 負責項目：網站首頁、前台整合、會員登入/註冊、會員資料/密碼修改。
- 使用技術：JAVA / JSP+Servlet / CSS / Bootstrap / JavaScript / MySQL / JDBC / MVC開發。
